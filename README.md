# 软件下载中心

一个简洁美观的软件下载中心，提供收银软件、驱动程序、工具软件等一键下载服务。

## 功能特点

- 🎨 **现代化UI设计**：采用卡片式布局，支持明暗主题切换
- 📱 **响应式设计**：完美适配桌面端和移动端设备
- 🌗 **主题切换**：支持浅色/深色主题，自动根据时间切换
- 🚀 **快速访问**：分类清晰，一键下载所需软件
- 🔗 **开发者链接**：点击顶部标题三次可进入开发者链接页面（彩蛋功能）

## 使用说明

1. **直接打开**：双击 `index.html` 即可在浏览器中使用
2. **本地部署**：可使用任何静态服务器部署（如 nginx、apache 等）
3. **主题切换**：点击右上角主题按钮切换明暗主题
4. **下载软件**：点击对应的软件卡片即可下载
5. **访问开发者链接**：在首页点击顶部"软件下载中心"标题三次（2秒内）

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计，使用 CSS 变量实现主题切换
- **JavaScript**：交互功能（主题切换、彩蛋跳转等）
- **无依赖**：纯静态页面，无需任何框架或库

## 特色功能

### 1. 主题切换
- 支持浅色/深色两种主题
- 可手动切换或根据时间自动切换（6:00-18:00 为浅色主题）
- 主题偏好保存在本地存储中

### 2. 彩蛋功能
- 点击页面顶部"软件下载中心"标题三次
- 2秒内完成点击即可进入开发者链接页面
- 完全隐藏，没有视觉提示

### 3. 外部链接标识
- 外部链接卡片使用紫色边框标识
- 在新标签页中打开，不影响当前浏览

### 4. 二维码提示
- 部分软件支持手机端下载
- 鼠标悬停在手机图标上可查看二维码

## 维护说明

### 添加新软件
在对应的分类中添加新的 `<a>` 标签：

```html
<a href="软件路径" class="download-card">
    <span class="card-icon">图标</span>
    <span class="card-title">软件名称</span>
    <span class="card-version">版本号</span>
</a>
```

### 修改下载链接
直接修改对应卡片的 `href` 属性即可

### 添加新分类
复制现有的 `<section>` 结构，修改分类标题和内容

## 注意事项

1. 所有下载链接需要确保文件路径正确
2. 外部链接建议添加 `target="_blank"` 属性在新窗口打开
3. 大文件下载建议提供文件大小信息
4. 定期检查和更新软件版本
